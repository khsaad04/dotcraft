# Tread
Simple template generator and dotfiles manager.

You can look at my [dotfiles](https://github.com/khsaad04/dotfiles) for an extensive example on how to use it to manage dotfiles.

```console
$ tread --help
Simple template generator and dotfiles manager

Usage: tread [OPTION] <SUBCOMMAND>

Options:
    -m, --manifest <FILE>  Path to Manifest file [default: ./Manifest.toml]
    -h, --help             Print help

Subcommands:
    sync      Symlink files and generate templates 
    link      Symlink files
    generate  Generate templates
```
